Abstract

This ontology defines more specific subclasses of terms from the wf4ever ontologies ro, wfdesc and wfprov, e.g. wf4ever:Image, wf4ever:WebServiceProcess in addition to associated properties for describing services and tools, e.g. wf4ever:command and wf4ever:script. This ontology can be used as part of annotations in an `wf4ever:WorkflowResearchObject`.

Download the wf4ever Ontology or import it from http://purl.org/wf4ever/wf4ever# in order to start using it in your work.
